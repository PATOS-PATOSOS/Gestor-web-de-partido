# Gestor-web-de-partido
Un pequeño club deportivo te encarga una página web sencilla para gestionar un partido amistoso.  La aplicación debe permitir:  Llevar el marcador del partido  Gestionar una lista de jugadores  Mostrar información clara y actualizada durante el partido  Debe ser una solución funcional, clara y bien estructurada, hecha con HTML5, CSS y JavaScript.
